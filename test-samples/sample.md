# DocMD Converter 테스트 문서

## 소개

이 문서는 **DocMD Converter**의 Markdown → DOCX 변환 기능을 테스트하기 위한 샘플입니다.

## 기능 목록

- DOCX를 Markdown으로 변환
- Markdown을 DOCX로 변환
- 최대 50개 파일 배치 처리
- ZIP 다운로드 지원

## 코드 예시

```javascript
function hello() {
  console.log("Hello, DocMD!");
}
```

## 표 테스트

| 항목 | 설명 | 상태 |
|------|------|------|
| 변환 엔진 | mammoth + turndown | 완료 |
| UI | Tailwind CSS | 완료 |
| 배포 | Vercel | 예정 |

## 인용문

> 모든 처리는 브라우저에서 이루어집니다.
> 서버로 파일이 전송되지 않습니다.
